# DOWNLOAD-2-VIA-DE-CONTAS-CAESB
DOWNLOAD 2º VIA DE CONTAS CAESB
Realizar o download da 2º via de contas da CAESB em um arquivo PDF.

Sobre este fluxo de automação:
Através de fluxo você pode realizar o download da 2 via da fatura no site da CAESB usando o numero da Conta

Vale lembrar que esse link para o download da 2 via fica ativo durante alguns dias do mês em que a conta esta disponível, por isso o fluxo possui uma gestão do erro caso o link não esteja disponível.

Para realizar esse download você deve ter em mãos os dados de:

Numero da Conta CAESB
Tipo: Web
Pre-requisitos: Fazer uso do Mozilla Firefox, Conta no Parceiro de ByPass de Catpcha DeathByCaptcha.
